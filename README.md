# Virtual Memory Analyser

Educational Flask + JS app to simulate page replacement algorithms (FIFO, LRU, MRU, Optimal).

## Run locally

```bash
pip install -r requirements.txt
python app.py
```

Open http://localhost:5000

## Notes
- Change `Frames` in UI and re-select algorithm (or press Reset) to ensure server uses the same frame count.
- Supported algorithms: FIFO, LRU, MRU, OPTIMAL.
